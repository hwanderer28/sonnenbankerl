# Resources

This file contains links to tools, data sources, APIs, and documentation relevant to the Sonnenbankerl project.

## Development Tools

- **Flutter**: Cross-platform UI toolkit for building mobile apps. [flutter.dev](https://flutter.dev/)
- **Dart**: Programming language used by Flutter. [dart.dev](https://dart.dev/)
- **PostgreSQL**: Open-source relational database. [postgresql.org](https://www.postgresql.org/)
- **PostGIS**: Spatial database extension for PostgreSQL. [postgis.net](https://postgis.net/)
- **QGIS**: Free and open-source geographic information system. [qgis.org](https://qgis.org/)
- **SolarQGIS**: Plugin for solar analysis in QGIS. [github.com/MateuszIlba/SolarQGIS](https://github.com/MateuszIlba/SolarQGIS)

## Data Sources
